# Picasso AI Chatbot

A modern desktop AI chatbot application with a sleek dark-themed interface, powered by Groq's Llama 3.3 70B model.
Just launch PicassoAI.exe and you can start chating!

## Features

- If you press Ctrl+Enter you'll get a short answer
- Real-time AI conversations using Groq API
- Automatic language detection and response
- Clean, modern dark UI design
- Threaded API calls for smooth performance
- Chat history with scrollable interface
- Custom visual elements and branding
